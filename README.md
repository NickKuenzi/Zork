# Zork

# Projektübersicht
Dieses Projekt ist eine textbasierte Adventure-Spiel-Implementierung im Stil von Zork, entwickelt als Teamprojekt in der IT-Lehre bei Benedict BVS. Es umfasst Räume, die durch Kommandos wie "go north", "back" oder "help" navigierbar sind, mit einer Konsolenoberfläche für das Gameplay.

# Features
-Mehrere verbundene Räume mit Beschreibungen und Ausgängen (N/S/E/W).
-Parser für Spielerbefehle (go, quit, help, back).
-Zurück-Navigation via Stack (previousRooms).
-Englische Raumnamen und Beschreibungen.
-Basierend auf Michael Köllings Originalvorlage, refactored für moderne Java

# Starte das Spiel: java ch.bbw.zork.Game.
Befehle eingeben:
go north (oder N/S/E/W).
back (zum vorherigen Raum).
help (verfügbare Kommandos).
quit (Spiel beenden).
Beispiel-Ausgabe: "You are in the foyer. Exits: north".
